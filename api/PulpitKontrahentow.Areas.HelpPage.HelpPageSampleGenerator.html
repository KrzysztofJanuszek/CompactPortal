GetSampleObject(Type)

Gets the sample object that will be serialized by the formatters. First, it will look at the SampleObjects. If no sample object is found, it will try to create one using PulpitKontrahentow.Areas.HelpPage.HelpPageSampleGenerator.DefaultSampleObjectFactory(PulpitKontrahentow.Areas.HelpPage.HelpPageSampleGenerator,System.Type) (which wraps an ObjectGenerator) and other factories in SampleObjectFactories.
